## Features

Time slice events:

* started - one-time
* in progress - periodic, every n minutes
* will elapse soon - repeated, warn me: 10, 5, 3, 1 minutes before time elapses
* has elapsed - one-time

Actions:

* notify via the console with elapsed time in minutes:seconds
* notify via Growl
